# Interactive To-Do List Project

This is a project for creating an interactive to-do list using JavaScript.

## Project Structure

The project has the following files:

- `index.html`: The HTML file for the to-do list.
- `style.css`: The CSS file for styling the to-do list.
- `script.js`: The JavaScript file for implementing the interactive functionality of the to-do list.

## Project Requirements

The to-do list should have the following features:

- Display a list of tasks.
- Allow users to add new tasks.
- Allow users to mark tasks as completed.
- Allow users to delete tasks.

## Getting Started

To run the project, simply open the `index.html` file in a web browser.
